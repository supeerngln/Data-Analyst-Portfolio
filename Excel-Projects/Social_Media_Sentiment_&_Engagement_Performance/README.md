## Project Overview
This project provides a comprehensive analysis of over 700 social media posts and interactions across Instagram, Twitter, and Facebook (2010-2023). The primary objective of this analysis was to uncover how user sentiment correlates with engagement metrics and to identify peak activity windows to optimize content strategy.

## Technical Stack
* **Data Cleaning:** TRIM and TEXT
* **Analysis:** Pivot Tables and Data Modeling
* **Visualization:** Interactive Excel Dashboard

## Key Accomplishment
* Resolved over 700 instances of inconsistent categorical data to ensure 100% accuracy in Pivot Table reporting.
* Transformed raw timestamps into discrete hourly segments, identifying a significant spike during the 19:30-21:30 window.
* Consolidated a fragmented list of 280+ sentiments into a Sentiment Clean column for better visualization.

## Key Insights
* Engagement reaches its highest point at **19:30**, tailoring a total of **2,153** interactions, proving that audience receptivity is highest in the mid-evening.
* The **USA** is the most active geographic segment with **188** unique interactions, followed closely by the **UK**.
* **Positive**,**Joy**, and **Excitement** are the primary drivers of high-volume engagements.
